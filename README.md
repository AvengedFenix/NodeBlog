# NodeBlog
First mini project using ExpressJS, EJS and MongoDB to create a blog

## Frameworks and dependencies
* ExpressJS (Back-end)
* EJS (Views)
* MongoDB (Database)
* Mongoose (ODM)
* Morgan

### Database hosting
MongoDB Atlas
